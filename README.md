# learn_fastapi_auth-project

## 项目概述

这是一个用于学习 FastAPI + Authentication 的实践项目. 项目采用 [FastAPI](https://fastapi.tiangolo.com/) 作为后端框架, 使用 [fastapi-users](https://fastapi-users.github.io/fastapi-users/latest/) 库处理用户认证, SQLAlchemy 作为 ORM 连接本地 SQLite 数据库. 前端部分使用原生 HTML/CSS/JavaScript, 不依赖现代前端框架 (如 Vue/React), 以便更好地理解前后端交互的基本原理.

## 技术栈

**后端:**
- FastAPI - Python Web 框架
- fastapi-users - 用户认证库
- SQLAlchemy - ORM 数据库框架
- SQLite - 本地数据库

**前端:**
- HTML/CSS/JavaScript (原生, 无框架)

## 开发方式

大部分代码将通过 AI (Claude Code) 辅助完成, 项目主要用于学习和实践目的.

## 项目目标

- 理解 FastAPI 的认证机制
- 掌握 SQLAlchemy ORM 的使用
- 学习前后端分离架构的基本实现
- 实践用户注册, 登录, 会话管理等功能
